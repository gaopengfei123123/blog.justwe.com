转 mac上一些好玩的命令行工具

2019-01-18
tools

mac上一些有意思的命令行小工具

asciinema 和 svg-term

assiiname 是用来命令行录屏的, svg-term 将文件转换成svg动图, 这个做截屏很方便

安装:

1
2

brew install asciinema
npm install -g svg-term

使用方式:

1
2

asciinema rec cast.json   // 开始录制, 数据保存在cast.json, 按 ctrl+d 保存
cat cast.json | svg-term-cli   //输出svg文件

比如下图 缺点就是如果文件特别大帧率受到严重的影响

sl 和 cmatrix 小特效玩具

安装

1
2

brew install sl
brew install cmatrix

平时装哔用的

bat

安装

1

brew install bat

如标题, 清理磁盘, 直接运行就行

安装

1

brew install ncdu
