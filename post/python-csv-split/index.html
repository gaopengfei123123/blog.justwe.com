是用脚本分割csv文件

2019-11-11
tools

1

pip install pandas

创建脚本:

1
2
3
4

import pandas as pd 
rows = pd.read_csv("csvfile.csv", chunksize=5000000) # 每个文件的尺寸, 以及导入文件名
for i, chuck in enumerate(rows): 
    chuck.to_csv('out{}.csv'.format(i)) # 输出文件名

运行

1

python demo.py
